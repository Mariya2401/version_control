# Подсказка по гиту

## Инициализация репозитория

```sh
git init
```

## Статус репозитория

```sh
git status
```

## Информация по работе с удаленными репозиториями

## Сервис для работы с удаленными репзиториями

```sh
GitHub
```

## Копирование удаленного репозитория к себе

```sh
git clone <адрес с сервиса GitHub>
```

## Отправить локальные изменения в удаленный репозиторий

```sh
git push
```

*Примечание: перед первым пушем нужно будет "подружить" наш git с GitHub. Система подскажет, как это сделать*

## Взять из удаленного репозитория все изменения (также будет сделан merge)

```sh
git pull
```

## Сделать копию чужого репозитория на своем аккаунте (на GitHub)

```sh
fork
```

*Примечание: при внесении изменений в чужой проект создаем новую ветку

## Сравнить и отправить изменения изначальному хозяину репозитория (на GitHub)

```sh
Compare&pull request
```

## Еще полезные команды

```sh
git remote - список настроенных удаленных репозиториев
git remote show <origin> - просмотр информации о данном репозитории
git remote rename <старое имя><новое имя> - переименовывание удаленного репзитория
git push --set-upstream origin <имя ветки> - добавление новой ветки в удаленный репозиторий
git push origin --delete <имя ветки> - удаление ветки на удаленном репозитории
```
